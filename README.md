# Student Enrollment Portal API

A production-ready REST API for managing university student enrollments, courses, and schedules. Built with FastAPI and PostgreSQL, this project demonstrates scalable backend architecture, secure authentication, and a modern CI/CD deployment pipeline.

## Features

- **Robust REST Architecture**: Full CRUD endpoints for Students, Courses, and Enrollments.
- **Secure Authentication**: JWT (JSON Web Token) based authentication with Bcrypt password hashing.
- **PostgreSQL Database**: Relational data modeling using SQLAlchemy ORM.
- **Data Validation**: Strict request/response validation using Pydantic schemas.
- **AI Integration**: Integrated an AI Academic Advisor using Groq and the Llama 3 model for instant course recommendations.
- **Test-Driven Design**: Pytest suite using isolated, in-memory SQLite databases for rapid testing.
- **Dockerized**: Fully containerized using Docker and `docker-compose` for reproducible environments.
- **CI/CD Ready**: Automated testing pipeline configured via GitHub Actions, and Infrastructure-as-Code deployment via Render.

## Tech Stack

- **Framework**: FastAPI (Python 3.12)
- **Database**: PostgreSQL (Supabase), SQLAlchemy
- **Authentication**: JWT, Passlib, Bcrypt
- **Testing**: Pytest, FastAPI TestClient
- **Containerization**: Docker
- **AI**: Groq API (Llama 3)
- **Hosting**: Render

## Getting Started

### Prerequisites
- Python 3.12+
- Docker (Optional, but recommended)
- A Supabase account (for PostgreSQL)
- A Groq account (for AI API key)

### Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/shreenxdhi/student-enrollment-api.git
   cd student-enrollment-api
   ```

2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Create a `.env` file in the root directory and add your keys:
   ```env
   DATABASE_URL=postgresql://postgres:[PASSWORD]@db.[PROJECT-REF].supabase.co:5432/postgres
   SECRET_KEY=your_super_secret_jwt_key
   ACCESS_TOKEN_EXPIRE_MINUTES=30
   GROQ_API_KEY=your_groq_api_key
   ```

5. Run the server:
   ```bash
   uvicorn app.main:app --reload
   ```

6. Open `http://localhost:8000/docs` to view the interactive Swagger UI.

### Running with Docker

Make sure Docker Desktop is running, then execute:
```bash
docker-compose up -d --build
```
The API will be available at `http://localhost:8000`.

### Running Tests

To run the automated test suite using the isolated SQLite database:
```bash
pytest -v
```

## API Documentation

The complete, interactive API documentation is automatically generated by FastAPI and available at `/docs` when the server is running. Key endpoints include:

- `POST /auth/register` - Register a new admin user
- `POST /auth/login` - Authenticate and receive a JWT
- `GET /students/` - List all students (Requires Auth)
- `POST /enrollments/` - Enroll a student in a course (Requires Auth)
- `POST /advisor/chat` - Talk to the AI Academic Advisor (Requires Auth)
